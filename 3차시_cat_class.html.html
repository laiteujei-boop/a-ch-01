1단계: 배경 그림 그리기

마법의 주문으로 배경 만들기 ✨

AI에게 앱 배경으로 쓸 예쁜 그림을 그려달라고 할 거예요. 아래 주문을 복사해서 duck.ai 채팅창에 붙여넣으세요.

초등학생 여자아이들이 좋아할 만한 귀엽고 사랑스러운 고양이 테마의 컴퓨터 바탕화면 배경 이미지를 그려줘. 파스텔 핑크와 보라색 구름, 반짝이는 별, 하트, 그리고 귀여운 아기 고양이 일러스트들이 패턴처럼 부드럽게 들어가 있으면 좋겠어. 너무 복잡하지 않고 포근한 느낌으로 부탁해.

2단계: 코딩 주문하기

고양이에게 생명 불어넣기 🧡

이제 진짜 앱을 만들 차례예요! 아래 주문을 복사해서 duck.ai에 붙여넣으세요.

🔥 주의: 붙여넣기 한 다음, [여기에 아까 복사한 이미지 주소를 붙여넣으세요] 이 부분을 지우고, 아까 복사했던 진짜 이미지 주소(Ctrl+V)를 넣어야 해요!

나는 '나만의 고양이 집사 다이어리' 앱을 만들고 싶어. [디자인] 1. 배경 이미지는 이 주소를 사용해줘: [여기에 아까 복사한 이미지 주소를 붙여넣으세요] 2. 앱 중앙에 아주 큰 귀여운 고양이 이모지(🐱)를 보여줘. 3. 그 아래에 둥글고 파스텔톤인 버튼 3개를 만들어줘: '츄르 주기 🐟', '장난감 놀이 🎣', '쓰다듬기 👋'. 4. 맨 위에는 '💖 행복 지수: 0' 점수를 보여줘. [동작] 1. '츄르 주기' 버튼을 누르면 고양이 이모지가 '냠냠! 최고다냥! 😻'이라고 말풍선을 띄우고, 행복 지수가 10점 올라가게 해줘. 2. '장난감 놀이'를 누르면 '우다다다! 신난다냥! 💨'이라고 말하고, 행복 지수가 15점 올라가게 해줘. 3. '쓰다듬기'를 누르면 '골골골... 기분 좋다냥 🧡'이라고 말하고, 행복 지수가 5점 올라가게 해줘. 4. 버튼을 누를 때마다 고양이 이모지가 귀엽게 통통 튀는 애니메이션을 넣어줘. [코드] HTML, CSS, 자바스크립트가 모두 합쳐진 하나의 완벽한 HTML 파일 코드로 작성해줘. 폰트는 귀여운 스타일로 해줘.

3단계: 앱 완성하기

나만의 앱 저장하기 (제일 중요!) 💾

AI가 만들어준 긴 영어 코드(설계도)를 메모장에 옮겨 적으면 끝!

  1. duck.ai 채팅창의 코드 상자 위에 있는 [Copy] 버튼 클릭!
  2. 메모장에 붙여넣기 (Ctrl + V)
  3. 메모장 메뉴에서 [파일] > [다른 이름으로 저장]
  4. 파일 이름은 꼭 cat.html 이라고 적어주세요. (뒤에 .html 필수!)
  5. 저장 위치는 바탕화면!

바탕화면에 생긴 cat 파일을 실행해서 귀여운 고양이를 만나보세요! 🎉
